Ice Cream

This is a hover effect. Some phones allow you to use long press to start a permanent hover and you could refresh the page to go back to the start position but really you need a mouse or similar device to trigger this properly.

Dynamic Changes

Choose an alignment

Here's some new CSS functionality that wasn't possible a few months ago without using JavaScript.

There's a wrapping div around the selectors box and the red field which holds a custom property. The custom property is used in the red field to set the alignment of the items within. By using the :has() selector, selecting a radio button will give the custom property that button's setting.

The wrapper is actually inside another new feature, the encompassing section is also a container. The size of the red field is set using container sizes and the layout will change depending on the size of the container.
